Parts Hotspot OCR v4.21 portable - YOLO main

Запуск:
1. Откройте папку PartsHotspotOCR.
2. Дважды щелкните PartsHotspotOCR.exe.

Python и интернет для запуска не требуются.
Модели YOLO уже включены в папку models.
YOLO + RapidOCR используются как основной поиск номеров.

Требования: Windows 10/11 x64.
Tesseract не включен. Приложение работает без него, но при отдельной установке
Tesseract получает дополнительный резервный способ OCR.
